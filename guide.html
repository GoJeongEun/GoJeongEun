Step1. Color

  1. 01. Primary
    Primary Color 버튼, 활성 상태와 같은 UI 전체의 주요 구성 요소에 사용된다.
  2. 02. Secondary color
    Secondary color 필터, 칩과 같이 UI에서 보조적인 역할을 하는 구성 요소 전반에 사용된다.
  3. 03. Grayscale
    Grayscale은 정보전달과 위계를 나타내는 중요한 역할을 한다. 주로 BG, Text, line에 사용되며 특히 텍스트는 가독성을 고려해야 하기 때문에 배경과의 명암비를 고려해야 한다.
  4. 04. Alpha
    다양한 배경 색상과 콘텐츠

  • ⭐ animate aos option
    data-aos
    사용할 애니메이션 효과명
  • data-aos-easing
    애니메이션 시간흐름에 따른 속도설정 옵션(default:ease)
  • data-aos-anchor
    특정한 객체에 anchor를 설정하여, 어떤 객체를 기준으로 애니메이션이 시작 될지를 설정
  • data-aos-delay
    애니메이션 재생 대기시간 설정(default:0)
  • data-aos-offset
    효과가 시작될 객체의 위치 설정
  • data-aos-duration
    재생시간 설정(default:400)
  • data-aos-anchor-placement
    어느 위치에서부터 적용될지 설정(default:top-bottom)
  • data-aos-once
    스크롤할때마다 재생될지를 설정(default:true)
    false로 스크롤 위로 올렸을 때 다시 애니메이션 재생

    - source 요소는 미디어 타입이나 코덱의 지원여부, 미디어 쿼리에 따라 브라우저가 선택할 수 있는 대체 비디오/오디오/이미지 파일을 명시할 수 있도록 해줍니다.
