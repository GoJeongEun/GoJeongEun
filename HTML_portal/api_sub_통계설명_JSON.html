통계설명

통계설명 API를 이용하기 위한 요청변수, 출력정보, 샘플데이터를 제공합니다. 출력포맷으로 JSON과 XML을 제공합니다.

  • 요청URL
    https://kosis.kr/openapi/statisticsList.do?method=getList
  • 요청변수
    요청URL로 요청변수, 변수타입, 설명, 비고를 제공합니다
    요청변수 변수타입 설명 비고
    apiKey

    * orgId(기관ID)+tblId(통계표ID)로도 가능

    		 String 발급된 인증 key 필수
    vwCd String 서비스뷰 코드
    · MT_ZTITLE : 국내통계 주제별
    · MT_OTITLE : 국내통계 기관별
    · MT_GTITLE01 : e-지방지표(주제별)
    · MT_GTITLE02 : e-지방지표(지역별)
    · MT_CHOSUN_TITLE : 광복이전통계(1908~1943)
    · MT_HANKUK_TITLE : 대한민국통계연감
    · MT_STOP_TITLE : 작성중지통계
    · MT_RTITLE : 국제통계
    · MT_BUKHAN : 북한통계
    · MT_TM1_TITLE : 대상별통계
    · MT_TM2_TITLE : 이슈별통계
    · MT_ETITLE : 영문 KOSIS
    		 필수

    예) &statId=통계조사ID 또는
    &orgId=기관ID&tblId=통계표ID
    parentId String 시목록 ID 필수
    format String 결과 유형(json) 필수
  • 출력결과
    출력변수로 출력변수,내용,설명을 제공합니다
    결과변수 설명
    VW_CD 서비스뷰ID
    VW_NM 서비스뷰명
    LIST_ID 목록ID
    LIST_NM 목록명
    ORG_ID 기관코드
    TBL_ID 통계표ID
    TBL_NM 통계표명
    STAT_ID 통계조사ID
    SEND_DE 최종갱신일
    REC_TBL_SE 추천 통계표 여부
  • 통계자료 예제
    (JSP, R, Python)
  • 샘플데이터 확인
    파라미터(Input)
    파라미터(Input)로 필드명,설명,조회조건을 제공합니다
    요청변수 설명 조회조건
    통계조사 ID statId
    요청항목 request
    결과(Output)
    서비스뷰 목록ID 목록명 기관코드 통계표ID 통계조사ID 최종갱신일 추천 통계표 여부
    MT_ZTITLE A_4 수산물 검역 및 수출검사 통계
    MT_ZTITLE A_7 주요 양식어종[8품종] 폐사 동향
    MT_ZTITLE A_3 인구동향조사
    MT_ZTITLE A_6 장래인구추계
    MT_ZTITLE A_1 국내인구이동통계
    MT_ZTITLE A_9 출입국자및체류외국인통계
    MT_ZTITLE A_8 지방자치단체외국인주민현황
    MT_ZTITLE A_5 장래가구추계
    MT_ZTITLE A_2 국제인구이동통계
    MT_ZTITLE A_001 고령친화용품제조업실태조사
